Descarregar o Opencv 2.4.13 no seguinte link:

https://sourceforge.net/projects/opencvlibrary/files/opencv-win/2.4.13/opencv-2.4.13.exe/download

Descompactar o OpenCV para uma pasta opencv no disco C;

Adicionar a seguinte entrada à Path nas variaveis de ambiente de sistema (não nas do utilizador):

C:\opencv\build\x64\vc12\bin









NOTA: No Visual Studio 2015 não é necessário alterar nada pois librarias ja foram adicionadas.
